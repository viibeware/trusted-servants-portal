{# Lightweight, dependency-free image lightbox. Usage: drop this partial onto any frontend page that wants to make images zoomable, then add `data-lightbox-scope` to one or more container elements. The script auto-discovers every inside those scopes (skipping images wrapped in so card thumbnails still link as expected) and wires a click → open-modal handler. Multiple images in the same scope get prev/next navigation; a single image hides the arrows. Closes on backdrop click, the X button, or Escape. Honors prefers-reduced-motion. The full-size source defaults to ``img.src``; admins can override with ``data-lightbox-src="..."`` on a per-image basis (e.g. to point at an unrescaled original when the page is showing a thumbnail). #}
